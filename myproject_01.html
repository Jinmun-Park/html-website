Youtube Analysis - Google API

  • You can enjoy searching trendy videos in any region ! By registering your Google API, you can enjoy using package to have a deeper analysis on any channel you have chosen.

  • There are several numeric figures you can add in your analysis such as like, dislike, views and number of comments. Your analysis can be more precise if you run anomaly dection in my code.

  • Hot or worst videos will be selected using anomaly detection. Finally, You can extract all the comments for your sentiment analysis.
    • Github Repository

    Are you interested how others think about your favoirite youtubers?

    Google API allows you to extract public data which we can select youtube channel and video ID for our analysis. The question is, how we design our coding layout to extract channel ID and draw several plots. One of the goals in this project is to use BERT(Bidirectional Encoder Representations from Transformers) machine learning technique to mark sentiment scores and visualize it. My final goal is making website using flask to provide a better interface for end-users to have fun in analyzing their favorite youtbers without cost.

    Description

    This is my first personal project using Python and Flask web-app. IsolationForest will be using for anomaly detection and BERT for sentiment analysis.

    This project just came into my mind mystery accident happened to Son Jeong Min in South Korea. The case of 22-year-old Son JM has been making constant headlines across the nation due to the growing mystery behind his disappearance and death. While the case seemed to be accidental upon first glance, numerous signs began to show that something was not right with Son’s death.

    Famous korean show 'Unanswered Questions' conducted various experiments with the advice of experts and recreated the possible situations during the night of the incident. According to the show, there is substantial evidence that Son Jung Min could have possibly walked into the water in an attempt to get sober while being heavily intoxicated.

    My heart goes out to his family during this difficult time. However, I have no intention to be side with anoyone but i just hope the case to be solved out clearly. The controversial incident like this has more diverse opinions and it has value in analyzing them as a statistcal perspective. My motivation of this work is to build class function for people to run easy sentiment analysis on any youtube comment.

    I have interesting result from Youtube Soccer channel during champions league final, 30th May 2021 and i will my update results soon. I hope everyone find this work interesting. I am open to hear any feedback from you all to improve my skills better.
